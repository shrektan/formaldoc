# FormalDoc Release Notes

## v1.18.0 — 2026-04-01

### 新功能

- **脚注支持**：Markdown 脚注（`[^1]`）自动转换为 Word 原生脚注，显示在页面底部，支持多个脚注和丰富内容格式

### 修复

- ListParagraph 和 BlockQuote 样式增加显式两端对齐

---

## v1.17.1 及更早版本

FormalDoc 从 v1.0 到 v1.17.1 的主要功能积累：

### 核心功能

- **Markdown 转 Word**：将 Markdown 转换为格式规范的 .docx 文档，使用 Word 样式方便二次编辑
- **8 套文档模板**：中文（公文 GB/T 9704、通用、学术、报告）和英文（标准、商务、学术、法律）
- **LaTeX 公式**：自动转换为原生 Word 公式（OMML 格式），在 Word 中可直接编辑
- **智能粘贴**：从 AI 聊天工具复制内容，自动转换为 Markdown
- **中英文引号转换**：一键将英文引号转为中文引号
- **CLI 工具**：命令行批量转换，支持 `--template` 和 `--title-level` 参数
- **npm 包**：可作为 Node.js 库集成到其他项目
- **国际化**：中英双语界面
- **移动端适配**：响应式布局

### 技术特点

- 全客户端处理，无需上传文件，保护隐私
- 支持表格、列表（多级嵌套）、引用块、删除线、超链接等丰富格式
- 模板样式可自定义（字体、行距、缩进等）
