# FormalDoc

**FormalDoc** helps you turn AI-generated text into properly formatted Word documents.

Paste the content generated by an AI model (Markdown supported), choose a Word template, and generate a clean, formal document in one click.

---

## How It Works

1. Copy the text generated by an AI model  
2. Paste it into FormalDoc  
3. Choose a built-in template or your own Word template  
4. Click **Generate**  
5. Download a ready-to-use Word document

No manual formatting in Word.
